# VMware Installation

## Objective
To install VMware Workstation for creating virtual machines.

## Steps Followed

1. Opened a web browser  
2. Searched for VMware Workstation download  
3. Downloaded VMware Workstation from the official website  
4. Opened the downloaded installer file  
5. Clicked on "Next" to start installation  
6. Accepted the license agreement  
7. Selected custom (manual) configuration instead of default settings  
8. Chose installation location and allocated required disk space manually  
9. Clicked on "Install"  
10. Waited for the installation to complete  
11. Clicked on "Finish"  

## Verification

- VMware application opened successfully  
- Main dashboard is visible  

## Outcome

VMware Workstation installed successfully with manual configuration and ready to create virtual machines.
